## How to Contribute

1. Fork the repo
2. Clone locally
3. Create a new branch (`feature/<your-feature>`)
4. Make changes and commit
5. Push branch and create a Pull Request
6. All contributions are welcome!

### Labels

- `good first issue` → Beginner-friendly
